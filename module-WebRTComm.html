WebRTComm

The JavaScript Framework WebRTComm allow Web Application developers to easily integrate multimedia communication service (e.g. VoIP) in their web site, thanks to the W3C WebRTC API. The WebRTComm Framework provides a high level communication API on top of the opensource JAIN SIP JavaScript Stack (implementing transport of SIP over WebSocket). By using a convergent HTTP/SIP Application Server (e.g. Mobicents MSS) or directly access a SIP server (e.g. Asterisk), the web developer can rapidly and easily link his web site to a telephony infrastructure.
